# data/ — Dados

Datasets e dados mockados usados pelo App BiT.

## Conteúdo sugerido

- **Dataset Vísent CDRView** — concentração de pessoas por zona + cobertura de rede ERB
  (5G/4G/3G) com coordenadas reais de antenas (Anatel). Baixe do repositório oficial:
  https://github.com/wongola-bit/appbit (inclui dicionário de colunas).
- **Mocks** — trilhas, vagas e perfis de exemplo para o MVP.

Detalhes de uso em [../docs/dataset-cdrview.md](../docs/dataset-cdrview.md).

## ⚠️ Atenção

- Arquivos grandes de dados **não** devem ir para o git — referencie a fonte ou use
  `data/` apenas para amostras pequenas.
- **Nunca** versione dados pessoais reais de usuários.
